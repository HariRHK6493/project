Cloning a repository:
Create a online github account. 
Command to clone a repository:
1. git clone <repository URL>// example - git clone https://github.com/HariRHK6493/project.git

For accessing different branches:
1. git checkout branchname

For modifications:
Edit the code and save the changes.
Commands:
1. git add . or filename or * 
2. git commit -m "message/comment"
3. git pull 
4. git merge
5. git push origin master/branchname
6. git status
